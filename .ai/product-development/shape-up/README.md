# Shape Up

Shape Up is a product development methodology where work is shaped before it's built, bet on in focused cycles, and shipped by autonomous teams within a fixed time budget. The core principle: **fixed time, variable scope** — set the appetite first, then design a solution that fits. With AI-assisted development, cycles are compressed to 1-week builds + 2-day cool-downs.
